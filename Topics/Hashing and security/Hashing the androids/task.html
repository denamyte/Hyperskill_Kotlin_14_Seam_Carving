Hashing the androids

Luke Skywalker wants to send a list of androids to the rebel band with a hash to ensure the integrity of the information due to the Empire's communication attacks.

We have a data class: data class Android(val name: String, val energy: Int).

Please complete fun calculateAndroidsHash(androids: List<Android>): String to obtain the hash using SHA-512. You must keep in mind that you must combine all the string values into one and then hash them.


Sample Input:
R2D2:100|C3PO:200

Sample Output:
